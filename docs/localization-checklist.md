# MindMate Localization Test Checklist

This checklist verifies 100% localization coverage for German (DE) and English (EN) modes.

## How to Test

1. Set language to German in Settings → Language
2. Navigate through ALL screens listed below
3. Verify NO English text appears anywhere
4. Repeat with English language setting
5. Mark each item ✓ when verified

## Terminal Commands for iPhone Testing

After exporting to GitHub and cloning locally:

```bash
# 1. Install dependencies
npm install

# 2. Add iOS platform (first time only)
npx cap add ios

# 3. Build the web app
npm run build

# 4. Sync to iOS
npx cap sync ios

# 5. Open in Xcode
npx cap open ios

# 6. Or run directly on connected device
npx cap run ios
```

For hot-reload testing during development:
```bash
# Terminal 1: Start dev server
npm run dev

# Terminal 2: Run on iOS (after initial sync)
npx cap run ios
```
